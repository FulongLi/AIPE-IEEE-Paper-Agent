# Zhengyu Lin IEEE Reference Collection

## Scope

This collection gathers IEEE-published reference candidates for Zhengyu Lin in power electronics, DC microgrids, embedded power-line/signaling communication, electric-vehicle charging and motor-drive applications, and related control topics.

## Source Method

- Primary discovery path: Crossref DOI metadata using `query.author=Zhengyu Lin`, Crossref ORCID filter `0000-0001-7733-2431`, and IEEE publisher filtering.
- Each included record has a DOI and an IEEE Xplore document URL from the Crossref `resource.primary.URL` field or DOI metadata.
- Direct IEEE Xplore page access from this environment returned HTTP 418, so the entries should be treated as DOI-verified and Xplore-link traceable, not as native IEEE-exported BibTeX.

## Record Count

- Included BibTeX records: 35.
- Entry keys use IEEE Xplore document/article numbers where available.

## Major Topic Clusters

- DC microgrid control, stability, droop control, impedance shaping, and power sharing.
- Power-electronic signaling, embedded communication, and power/data dual modulation.
- Energy routers, DC grid clusters, and peer-to-peer electricity trading support.
- EV charging, wireless power/data transfer, EV powertrain modeling, and multi-phase machine converter integration.
- PV maximum-power-point tracking and PV optimizer communication.
- Motor-drive and compressor current-control applications.

## Verification Notes

- All included records have DOI metadata and an IEEE Xplore document URL.
- IEEE Xplore direct scraping was blocked during collection; final manuscript use should still verify the exact citation export from IEEE Xplore when browser access is available.
- The 2025 DDCLS paper `10.1109/DDCLS66240.2025.11065636`, titled "Key node-based Observable and Predictable Technology for Distributed PV Clusters Research", was found by exact-name search but excluded from this collection because its solo author affiliation and topic suggest a likely homonym rather than the same power-electronics researcher.

## Known Limitations

- This is an IEEE-focused collection, not a complete Zhengyu Lin bibliography.
- Some relevant non-IEEE/IET/MDPI/Springer records found by ORCID are intentionally not included here.
- Page fields and capitalization follow Crossref metadata where IEEE Xplore export could not be accessed.
